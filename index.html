Projects

Smart Vision Automatic Attendance Tracker (2025)

Smart Vision Automatic Attendance Tracker is an AI-powered system that uses real-time facial recognition to automate session wise attendance in educational Institutions. It integrates Python, OpenCV, Tkinter, and SQLite to capture, recognize, and store attendance data efficiently. Designed with a user-friendly GUI, it enhances accuracy, prevents proxy attendance, and streamlines record management.

Wi-Fi Controlled Toy Car using Arduino (2023)

This project involves a toy model car that can be controlled via a mobile application. By establishing a WiFi connection between the car and the mobile device, users can easily move the car forward, backward, left, or right. The system is powered by an Arduino, making it a fun and interactive way to explore wireless control technology.
